### Prerequisites
- **Node.js 16+** for React frontend

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/newsinsight.git
cd newsinsight
```

### 2. Backend Setup
```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements-backend.txt

# Configure environment
cp .env.example .env
# Edit .env with your API keys and AWS configuration
```

### 3. Frontend Setup
```bash
# Install React dependencies
npm install

# Start development server
npm start
```

### 4. Start Backend
```bash
# In a separate terminal, with venv activated
python backend.py
```

### 5. Access Application
- **Frontend**: http://localhost:3002 (or next available port)
- **Backend API**: http://localhost:8000
- **API Documentation**: http://localhost:8000/docs

## ğŸ“‹ Configuration

### AWS Setup
1. **Configure AWS CLI**: `aws configure --profile newsinsights`
2. **Create DynamoDB table**: `news_metadata`
3. **Create S3 buckets** for processed and raw data
4. **Enable Bedrock** model access (Claude recommended)

### API Keys Required
- **NewsAPI**: Get free key at https://newsapi.org/register
- **Guardian API**: Get free key at https://open-platform.theguardian.com/access/

### Environment Variables
Copy `.env.example` to `.env` and configure:
- AWS credentials and region
- DynamoDB table name
- S3 bucket names
- Bedrock model ID
- News API keys
